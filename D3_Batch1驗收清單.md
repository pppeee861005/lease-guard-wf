## 🎯 驗收核心原則

### 原則1：Original 欄位 - 零容忍
> **鐵律**：original 與官方 PDF 逐字比對，錯一字即打回

**檢查方式**：
- [ ] 將 original 欄位複製出來
- [ ] 對標官方內政部 PDF 原文
- [ ] 檢查是否有任何字詞改動（包括標點、空格）
- [ ] 檢查是否有遺漏或增加的內容

**不接受的情況**：
- ❌ 用「…」代替原文某段
- ❌ 改寫法律術語使其更「易讀」（plain 的責任是易讀，original 只負責準確）
- ❌ 拆分成多句（保持原始段落結構）

### 原則2：Plain 欄位 - 給19歲看得懂
> **標準**：零法律術語 + 具體例子 + 1-3句簡潔口語

**檢查方式**：
- [ ] 唸給一個沒租過房的人聽（自測法）
- [ ] 計數：出現了多少個法律術語（應該是 0 個）
- [ ] 檢查：是否有具體例子或數字對比
- [ ] 檢查：句數是否控制在 1-3 句（便於快速理解）
- [ ] 檢查：是否包含「AI起草，待驗收」標記

**高品質 Plain 範例**（電費第6項）：
```
「房東如果向你收電費，價格不能超過台灣電力公司當月的『平均電價』。
 絕對不能按『最高級距』來超收。」

品質指標：
✅ 零法律術語（沒有「當期每度平均電價」）
✅ 對比鮮明（平均電價 vs 最高級距）
✅ 句數適中（2句）
✅ 有具體提醒（「絕對不能」）
```

### 原則3：Violations 欄位 - 實務樣態
> **標準**：3-5 個真實或合理的違規樣態，每個包含 pattern + example

**檢查方式**：
- [ ] 樣態數量：是否在 3-5 個範圍
- [ ] Pattern 清晰度：描述違規行為是否一眼看懂
- [ ] Example 逼真度：舉的例子是否真實可信（最好基於 PTT 或判決書案例）
- [ ] Severity_flag：違規樣態的等級標記是否合理

**檢查項**：
```
高品質違規樣態：
{
  "pattern": "按『級距最高價』計費而非平均電價",
  "example": "「您用400度，我按每度3.68元計費（最高級距），共1,472元」
            但台電實際費用只要1,247.6元，超收224元",
  "severity_flag": "高"
}

檢查清單：
✅ Pattern 說的是「什麼違規」（不是「怎麼違規」）
✅ Example 帶著具體數字（讓讀者能計算損失）
✅ Severity_flag 與該項整體 severity 一致
```

### 原則4：Severity 欄位 - 決策邏輯
> **標準**：按 D2 定義的三層判斷標準執行

**高/中/低 判斷標準**（D2 已定版）：
```
🔴 高：年損失 ≥$12,000 或 不得記載項 或 難救濟 或 居住權威脅
   例子：電費超收($2,400-6,000/年) → 乘以3-5倍違約責任 = 超過$12,000
         承租人終止權限制 → 居住權威脅 → 自動為高
         安全瑕疵 → 難救濟 → 自動為高

🟡 中：年損失 $3,600-12,000 且 可救濟
   例子：押金延遲返還 → 可走法律程序救濟
         管理費不透明 → 可請求會計查核
         修繕延遲 → 可申請修繕費扣押金

🟢 低：年損失 <$3,600 或 程序瑕疵
   例子：審閱期不足 → 程序瑕疵但無實質損害
         文書未列清 → 程序不完整但不影響權益
```

**檢查方式**：
- [ ] 根據該項的主要違規類型，判斷是否符合高/中/低定義
- [ ] 有疑義的項目（例如多維度但無明確金額的），記入 `disputed.md`
- [ ] 檢查是否與 D2 抽查驗證的結論一致

## 🔴 重點項目驗收指南

### ⭐ 第6項（電費）- 最高風險項

**官方原文範圍**：
- 應記載「每度電費不得超過該租賃標的電費單『當期每度平均電價』」
- 涉及 109 年版與 114 年版的差異

**重點檢查**：
- [ ] Original 是否完整包含「當期每度平均電價」的定義
- [ ] Plain 是否清楚對比「平均電價」與「最高級距」的差異
- [ ] Violations 是否涵蓋常見的電費超收案例（$2,400-6,000/年）
- [ ] Severity 是否為「高」（年損失達$12,000邊界）

**關鍵數字驗證**（基於計劃14的電費指南）：
```
台灣電費級距（月用電度數）：
- 0-110 度：$2.66/度
- 111-330 度：$3.13/度
- 331-500 度：$4.39/度
- 500+ 度：$6.12/度

超收案例計算：
房客用 300 度 →
- 正確：110×2.66 + 190×3.13 = 292 + 595 = 887元
- 違規：300×3.13 = 939元 → 超收 52元/月 = 624元/年

高風險案例（最高級距計費）：
房客用 300 度按 6.12 元計 = 1,836 元（正確應為 887 元）→ 超收 949 元/月 = 11,388 元/年
```

### ⭐ 第18項（承租人終止權）- 114年新規重點

**官方原文範圍**：
- 114 年 4 月 18 日重大修正
- 刪除「致難以繼續居住」高門檻限制
- 直接列舉六種具體情形
- 新增第(六)款：繼承人終止權

**重點檢查**：
- [ ] Original 是否清楚標記出 114 年新修正的內容
- [ ] Original 是否完整列舉六種終止情形
- [ ] Plain 是否簡潔說明新法的核心變化（門檻降低）
- [ ] Violations 是否涵蓋舊法常見的違規（禁止提前終止、高額違約金）
- [ ] Severity 是否為「高」（居住權威脅）

**新舊法對比**（重點記載）：
```
舊法（109年）：「承租人因租賃物之瑕疵，致難以繼續居住者，得終止租約」
新法（114年）：六種具體情形（無「難以繼續」高門檻）

六種新情形：
1. 租賃物有影響安全或衛生之瑕疵
2. 租賃物有違反消費者保護法第 52 條之瑕疵
3. 出租人未為必要修繕（逾30天）
4. 室內裝修有危害安全之情形
5. 出租人有騷擾行為
6. 【新增】承租人過世，繼承人可終止（不限於配偶）
```

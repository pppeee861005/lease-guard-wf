#### 欄位4：plain（**易讀性欄位**）
- **型別**：String
- **值**：白話解釋
- **目標讀者**：19 歲，首次租屋，零法律背景
- **規則**：

  - ✅ 使用日常口語（「東西」代替「物品」、「修好」代替「修繕」）
  - ✅ 1-3 句簡潔陳述
  - ✅ 附加具體例子或數字對比
  - ✅ 結尾加註「AI起草，待驗收」
  - ❌ 不允許冗長段落
  - ❌ 不允許超過 3 句（閱讀疲勞）
  - ❌ 不允許定義而不示例
- **長度**：50-150 字
- **易讀度檢驗**：

  - 他會有具體的行動感嗎？(好的 plain 應該讓讀者有「我要檢查什麼」的感覺)

#### 欄位5：violations（**樣態列舉欄位**）
- **型別**：Array of Object
- **數量**：3-5 個樣態
- **結構**：每個樣態含 3 個子欄位
  ```json
  {
    "pattern": "違規樣態（『什麼』是違規）",
    "example": "具體案例（帶數字或實際情節）",
    "severity_flag": "高｜中｜低"
  }
  ```
- **Pattern 規則**（8-20 字）：
  - ✅ 說明「違規行為是什麼」
  - ✅ 與原文第8-13項的要求形成「反面」對應
  - ❌ 不說「怎麼違規」的細節
  - 例子：「禁止租屋者飼養寵物」vs「租屋者偷偷飼養超過契約限定數量的寵物」

- **Example 規則**（50-150 字）：
  - ✅ 基於真實案例（PTT 災難案例優先）或判決書
  - ✅ 含具體數字或時間（「超過3個月未修繕」「延遲賠償5萬元」）
  - ✅ 展現違規的實際傷害（「房客因漏水損失1萬元物品」）
  - ❌ 不允許純假設（「想象一下…」）
  - ❌ 不允許過度複雜（「假設…然後…結果…」，超過 3 層邏輯）

- **Severity_flag 規則**：
  - 應與該違規樣態的嚴重程度對應（不一定等於整項的 severity）
  - 例如：第9項（修繕責任）整體可能是「中」，但「房東完全不修」的樣態應標為「高」

**Violations 高品質範例**（第6項電費）：
```json
{
  "pattern": "按『級距最高價』計費而非平均電價",
  "example": "「您用400度，我按每度3.68元計費（最高級距），共1,472元」
            但台電實際費用只要1,247.6元，超收224元/月 = 2,688元/年",
  "severity_flag": "高"
}
```

#### 欄位6：severity（**風險等級欄位**）
- **型別**：String
- **值**：「高」、「中」、「低」
- **判斷標準**（基於 D2 定版）：

```
🔴 高 - High Risk
定義：直接金錢損失 ≥$12,000/年 OR 難救濟 OR 居住權威脅 OR 不得記載項
適用項：
- 第8項（使用限制）：如果有居住權威脅 → 高
- 第9項（修繕責任）：如果房東完全不修且危害安全 → 高
- 第10項（設備清冊）：通常是低-中
- 第11項（保險）：如果遺漏保險導致火災損失 → 高
- 第12項（訴訟費用）：如果不公平分配 → 中（可救濟）
- 第13項（其他約定）：看具體內容（可能跨越高-中-低）

🟡 中 - Medium Risk
定義：年損失 $3,600-12,000 AND 可救濟
適用項：
- 第9項（修繕責任）：房東延遲修繕 → 中（可走法律程序）
- 第12項（訴訟費用）：不合理的分擔 → 中（可爭議）

🟢 低 - Low Risk
定義：年損失 <$3,600 OR 程序瑕疵
適用項：
- 第10項（設備清冊）：程序要求，無實質損害 → 低
- 第13項（其他約定）：若只是備忘條款 → 低
```

**判斷流程**（決策樹）：
```
該項涉及「金錢損失」嗎？
  ├─ 是 → 預估年損失金額
  │  ├─ ≥$12,000 → 「高」
  │  ├─ $3,600-12,000 → 「中」
  │  └─ <$3,600 → 「低」
  │
  └─ 否 → 該項涉及「居住權」嗎？
     ├─ 是 → 「高」
     └─ 否 → 該項涉及「能否事後救濟」嗎？
        ├─ 難救濟 → 「高」
        ├─ 可救濟 → 「中」
        └─ 程序瑕疵 → 「低」
```

## 🎯 特殊項目重點指導

### 第9項（修繕責任） - 最複雜項目

**背景**：
- 涉及民法第 424-427 條的修繕責任
- 租賃契約常見的糾紛源頭
- 容易超出「應記載事項」範圍而進入「法律詮釋」

**Original 預期內容**：
```
應記載「承租人應保持租賃物內外之整潔，亮麗，並負責日常維修
 （如裂紋修補、油漆、小零件、玻璃及衛浴設備簡易維修等），
 出租人應負責租賃物之重大修繕或改修」
```

**Plain 編寫原則**：
- ✅ 清楚區分「房客負責」vs「房東負責」
- ✅ 用具體例子（「貼壁紙」vs「漏水」）
- ✅ 避免「民法 424 條」等法律術語
- ❌ 不要進入「什麼叫『重大』」的邏輯糾紛

**Violations 重點**：
- 房東推卸修繕責任
- 房東延遲修繕（超過 30 天）
- 房客被強迫承擔「不應該他負責」的修繕（如結構性漏水）
- 房東扣押金以修繕名義

### 第13項（其他約定事項） - 最開放項目

**背景**：
- 這是合約的「自由約定欄」
- 常見的陷阱項（高利貸般的違約金、無理限制等）
- 應記載的目的是「讓這一欄不成為法外之地」

**Original 預期內容**：
```
應記載「其他雙方約定事項：【此欄由雙方自由約定，但不得違反
 消費者保護法及其他相關法令】」
```

**Plain 編寫原則**：
- ✅ 強調「自由約定」但「有限制」
- ✅ 列舉常見的不公平條款（無理違約金、限制出入時間等）
- ✅ 提醒讀者「不合理的約定可以拒簽」
- ❌ 不要列舉所有可能的陷阱（太冗長）

**Violations 重點**：
- 違約金過高（年租金的多倍）
- 限制居住自由（不得朋友來訪、限制出入時間）
- 單方面解約權（只有房東可以解約，房客不行）
- 隱藏的費用（「管理費」「保險費」等額外收費）

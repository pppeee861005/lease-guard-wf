## 🔴 重點項目驗收指南

### ⭐ 第9項（修繕責任） - 最複雜項目

**重點檢查**：
- [ ] Original 是否清楚區分「房客負責」vs「房東負責」的修繕
- [ ] Plain 是否用具體例子（貼壁紙 vs 漏水）說明
- [ ] Violations 是否涵蓋：房東推卸責任、延遲修繕、房客被強迫承擔不應該的修繕
- [ ] Severity 判斷是否合理（完全不修且危害安全→高；延遲修繕→中）

### ⭐ 第13項（其他約定事項） - 最開放項目

**重點檢查**：
- [ ] Original 是否清楚說明「自由約定但有限制」的邏輯
- [ ] Plain 是否列舉常見陷阱（無理違約金、限制居住自由、單方面解約權）
- [ ] Violations 是否涵蓋高風險案例（年租金多倍的違約金、限制出入、隱藏費用）
- [ ] Severity 判斷是否根據具體內容判定（不一定都是高）

## ✅ 驗收三要素（記住這個）

驗收時只需檢查三點：

### 1️⃣ Original（官方原文）- 零容忍
```
✅ 逐字抄錄，零改動
✅ 保留所有標點和段落
✅ 不允許用「…」代替任何內容

快速檢查法：
複製官方PDF的原文 → 複製JSON的original → 逐字對比
是否完全相同？是 → 通過；否 → 記錄疑義
```

### 2️⃣ Plain（白話版本）- 給19歲看得懂
```
✅ 零法律術語（沒有「定著物」「動產」等）
✅ 使用日常口語（「東西」而非「物品」）
✅ 1-3句簡潔，附具體例子
✅ 結尾有「AI起草，待驗收」標記

快速檢查法：
大聲唸給沒租過房的人聽 → 他理解嗎？
是 → 通過；否 → 記錄疑義
```

### 3️⃣ Severity（風險等級）- 符合決策邏輯
```
✅ 按決策樹判定（見下方）
✅ 與該項性質相符
✅ 與 violations 樣態一致

快速判斷法：
涉及金錢損失≥$12,000/年？→ 高
涉及居住權威脅？→ 高
難救濟？→ 高
其他 → 中或低
```

## 常見問題

**Q: 我能信任這個工具嗎？**
A: 信任來自透明。查看 `eval/reports/` 看實際的評測數字，包括漏報和誤報。如果你看到漏報數據，說明我們在誠實回報，而非隱瞞。

**Q: 系統有法律責任嗎？**
A: 沒有。本工具只陳述「這些條款與官方規範有落差」，不提供法律意見。具體爭議請洽專業律師。

**Q: 我能在自己國家用這個嗎？**
A: 可以，通過 Fork。詳見 `fork-guide.md`（W4 發布）。最關鍵的步驟：
  1. 替換規則庫（適配當地法規）
  2. 更新轉介資源（找到當地的援助組織）
  3. 修改 OCR 語言設定
  4. 保留三大鐵律（不能改）

**Q: 這個項目會商業化嗎？**
A: 不會。代碼永遠開源，規則庫永遠免費。我們的商業邊界是明確的：工具免費，有償服務會是「為特定組織做定製化適配」（如某縣政府的勞動契約檢測系統），但核心代碼不變。

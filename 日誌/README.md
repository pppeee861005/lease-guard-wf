## 📝 日誌類型說明

### work_log_YYYYMMDD.md
**用途**：記錄每日工作進展
**內容**：完成任務、遇到問題、重要發現
**更新頻率**：每日
**例子**：`work_log_20260612.md`

### 決策記錄/
**用途**：記錄重要決策的理由和過程
**內容**：為什麼選擇這個方案、有什麼替代方案、未來可能的調整
**更新頻率**：按決策發生時間
**例子**：`決策記錄/為什麼選擇Subagent並行結構.md`

### 進度報告/
**用途**：週期性（通常每週）的進度總結
**內容**：完成度百分比、關鍵成就、下週計劃
**更新頻率**：每週
**例子**：`進度報告/週進度_20260609-20260615.md`

### 戰報/
**用途**：實戰應用時的詳細案例記錄
**內容**：真實租約的檢查過程、發現的問題、解決方案
**更新頻率**：應用成功後記錄
**例子**：`戰報/戰報_001_某房屋租約檢查案例.md`

## 🔍 檢索規則

### 按日期查找
```bash
ls work_log_*.md | grep "20260612"
```

### 按主題查找
使用 `/` 後綴分類，例：
```
work_log_20260612_電費條款修正.md
work_log_20260612_蜂群效率測試.md
```

### 按重要性查找
檔名前綴：
- 🔴 `urgent_` — 緊急待處理
- 🟡 `important_` — 重要但非緊急
- 🟢 `routine_` — 日常工作

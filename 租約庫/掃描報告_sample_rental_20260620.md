##### 🔴 Item 28-29 案例：稅賦轉嫁

**違規條文**：
```
第八條(1)
「承租人應按比例負擔房屋稅及地價稅增加部分（房屋稅率 1.2%）」
「如政府開徵新稅或提高稅率，承租人應全額負擔增加之稅賦」
```

**違規分析**：
- **違規特性**：將房東的稅務責任轉嫁給房客
- **法律依據**：住宅租賃定型化契約不得記載事項第28-29項
- **房客年損失估算**：
  ```
  房屋稅損失 = 28,000元/月 × 12月 × 1.2% ÷ 3年 ≈ 13,440元/年
  地價稅損失 = 未明確計算（可能另有損失）
  新稅增量 = 未知（房客承擔無限風險）
  ```
- **評級**：🔴 嚴重違規（違法無效）

**法律判斷**：房屋稅及地價稅是房東的責任，不能轉嫁給房客

---

##### 🔴 Item 30 案例：房東單方終止權

**違規條文**：
```
第九條(1)
「承租人有任何違約行為，甲方得不經告知及調解程序直接解除租約，並沒收全額押金」

第九條(3)
「甲方認為承租人不適合繼續租住，得隨時終止租約」
```

**違規分析**：
- **違規特性**：房東可無故無程序地趕房客走
- **法律依據**：民法第440條、租賃定型化契約第30項
- **實際影響**：
  ```
  • 房客失去租約穩定性
  • 房東可不給理由直接趕房客
  • 房客無調解機會
  • 租約形同虛設
  ```
- **評級**：🔴 嚴重違規（違法無效）

**法律判斷**：房東終止租約需符合法定理由（如欠租2個月、違法使用等），不能無故無程序

##### 🔴 Item 32 案例：違約金超額

**違規條文**（多個）：
```
第五條(2)
「中途退租：承租人如欲提前終止租約，應提前60日通知出租人，
  否則沒收全額押金84,000元，並賠償房東五個月租金140,000元」

第五條(2)
「未經許可改裝：承租人不得擅自改裝房屋內部（包括釘釘子），
  違者每次罰款10,000元」

第十條
「嚴禁轉租或出租房間。違者：沒收全額押金84,000元、
  支付違約金300,000元、甲方得立即終止租約」
```

**違規分析**：
```
法定上限 = 1個月租金 = 28,000元

實際違約金設定：
─────────────────────────────────
提前終止：84,000 + 140,000 = 224,000元
  超過倍數：224,000 ÷ 28,000 = 8倍（超過800%）

改裝罰款：10,000元/次
  超過倍數：10,000 ÷ 28,000 = 0.36倍（略低但有多次上限無明確規定）

轉租罰款：300,000元
  超過倍數：300,000 ÷ 28,000 = 10.7倍（超過1070%）
─────────────────────────────────
```

**房客實際風險**：
- 如提前1個月終止：需賠224,000元（8倍租金）
- 如釘5次釘子：需賠50,000元（1.8倍租金）
- 如經朋友同意短住轉租：需賠300,000元（10倍租金）

**評級**：🔴 極嚴重違規（違法無效）

**法律判斷**：違約金不能超過1個月租金，超過部分房客有權拒付

###### 2️⃣ 禁止訪客留宿 + 罰款

**條文**：第五條(3)(2)「禁止邀請訪客留宿 — 違者每次罰款5,000元」

**違反原因**：訪客自由是基本人權，不能限制

**評級**：🔴 高風險（人權侵害）

###### 3️⃣ 禁止任何裝飾

**條文**：第五條(3)(3)「禁止做任何裝飾 — 包括貼壁紙、掛畫、放置盆栽，違者罰款3,000元」

**違反原因**：合理布置自由是居住權的一部分，法律允許房客做合理改善

**評級**：🔴 高風險（生活自由受限）

---

###### 4️⃣ 禁止除濕機、冷暖氣

**條文**：第五條(3)(4)「禁止使用除濕機、冷暖氣機 — 違者罰款1,000元/次」

**違反原因**：生活設備使用自由，尤其冷暖氣是安全需求（夏天>38°C、冬天<5°C危害健康）

**評級**：🔴 高風險（安全威脅）

---

###### 5️⃣ 房東無通知進出 + 裝監視器

**條文**：第七條(3)
```
「甲方得不經事先通知，在任何時間進出房屋進行檢查、維修或其他目的」
「甲方得保留房屋備份鑰匙，承租人不得要求甲方歸還」
「甲方可在承租人不知情的情況下進行房屋檢查或安裝監視器」
```

**違反原因**：侵犯隱私權（憲法保障），法律要求房東進出須事先告知

**評級**：🔴 極嚴重（憲法級別人權侵害）

---

###### 6️⃣ 強制清潔費 + 額外扣款

**條文**：第十一條(9)
```
「搬家時之強制清潔：承租人須自行或聘請甲方指定之清潔公司清潔房屋
  清潔費用為15,000元（由甲方指定之廠商，費用不得議）
  甲方若認為清潔不符標準，得加收5,000元」
```

**違反原因**：
- 強制指定清潔廠商（違反自由選擇權）
- 費用超過必要返還原狀費用
- 甲方可單方認定「不符標準」加收（無明確標準）

**房客潛在損失**：15,000 ~ 20,000元

**評級**：🔴 高風險（強制消費）

---

###### 7️⃣ 額外雜費（多項）

**條文**：第十一條(4-6)
```
垃圾清運費：每月300元
房屋定期檢查費：每月100元（用於甲方定期檢查）
保險費：年費2,000元（強制購買租賃住宅火險）
```

**違反原因**：
- 垃圾清運費應由管委會負責或分攤，不應由房東單方收取
- 房屋檢查費名目不當（房東維護房屋是基本責任）
- 強制購買火險且指定費用（應讓房客自行投保）

**房客年度額外損失**：
```
垃圾清運費：300元 × 12月 = 3,600元/年
檢查費：100元 × 12月 = 1,200元/年
保險費：2,000元/年
小計：6,800元/年
```

**評級**：🔴 高風險（名目費用過多）
